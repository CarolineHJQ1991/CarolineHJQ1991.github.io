Github实践&Git Command

Github实践&Git Command

###1.创建本地仓库

  • 1 mkdir git-tutorial
  • 2 cd git-tutorial
  • 3 git init
  • 4 git status

###2.在仓库中创建一个文件 (本地git仓库操作)

  • touch README.md
  • git status 显示没有被跟踪
  • git add README.md 添加文件到仓库中
  • git status b被跟踪
  • git commit -m “description” 提交
  • git log 直到当前的提交日志
  • git log –pretty=short 显示精简日志
  • git log -p README.md
  • git diff 工作区和暂存区的区别
  • git diff HEAD 当前要提交和上次提交的区别
  • git checkout -b feature-A == (git branch feature-A git checkout feature-A) 创建特性A分支
  • git checkout master 切换分支
  • git merge –no-ff feature-A (要在主分支合并) 非快速提交,会有历史
  • git log –graph 图表显示
  • git relog 显示所有历史记录
  • git reset –hard comment id 回转到想要的历史提交
  • git commit –amend 修改提交信息
